# TFG: Sistema de Gestión de Plantilla - Estilo Inazuma Eleven

## Descripción del Proyecto
Este proyecto consiste en una aplicación **CRUD** desarrollada en **C# / .NET** (WPF) para el curso 25/26. El objetivo es administrar una plantilla de fútbol y sus roles inspirándose en el universo de Inazuma Eleven.

### Características principales:
* **Gestión de Plantilla:** Administración completa de jugadores y sus roles.
* **Entrenamientos y Partidos:** Creación y seguimiento de sesiones de entrenamiento y partidos.
* **Ejercicios:** Gestión de ejercicios que deben hacer los jugadores.
* **Personalización:** Registro de **supertécnicas** y formaciones de equipo.

### FUNCIONES NUEVAS A DESAROLLAR
* **Rediseño de UI:** Cambiar flujo de la aplicacion y apariencia para simular que el usuario esta operando con un movil y navegando entre aplicaciones.
* **"App" de banco:** Una pequeña "aplicacion" de banco con aspectos basicos relaciondos con la economia del usuario que inicie sesion.
* **Mercado de fichajes:** Un apartado que tenga un mercado de fichajes para poder vender y comprar jugadores de o para tu club.
* **Dependiendo del tiempo disponible se pueden implementar nuevas ideas**.

## Stack Tecnológico
* **Lenguaje:** C# / .NET (WPF)
* **Base de Datos:** MySQL
* **Control de Versiones:** Git / GitHub
